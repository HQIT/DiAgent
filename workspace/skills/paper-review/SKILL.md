# 论文审稿 Skill

供**审核员**使用，规范审稿维度与意见表述。

## 审核维度

- **结构与逻辑**：章节完整度、顺序、论证链条是否清晰。
- **内容与论证**：论点是否有据、是否缺文献/数据、是否重复或矛盾。
- **实验与结果**：设置是否清楚、结果是否充分、分析是否到位。
- **语言与格式**：表述准确、语病、引用与格式规范。

## 意见书写规范

- 按章节或维度分条列出，每条**具体可执行**（如「3.2 节补充与 [文献X] 的对比」）。
- 可区分「必须修改」与「建议优化」。
- 结尾可给总体评价与优先修改顺序，不代替作者重写。

## 使用建议

- 只输出审稿意见与建议，供主 agent 转交编写者修改。
