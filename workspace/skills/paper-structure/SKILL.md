# 论文结构 Skill

供**编写者**使用，规范论文整体结构与各章节写法。

## 推荐结构（可依领域微调）

- **摘要**：问题、方法、主要结果、结论（200–300 字）。
- **引言**：背景、动机、贡献点、全文结构。
- **相关工作**：分类综述、与本文的差异。
- **方法/模型**：假设、符号、算法或模型描述。
- **实验与结果**：设置、指标、结果表/图、分析。
- **讨论**：局限、扩展、启示。
- **结论**：总结贡献与未来工作。
- **参考文献**：统一格式（如 GB/T 7714 或 APA）。

## 使用建议

- 先定大纲再写各节，保证前后一致。
- 每节有明确主题句，段落间有递进或对比关系。
