# 实验设计 Skill

供**编写者**使用，规范实验部分写作与图表描述。

## 实验部分建议结构

1. **实验设置**：数据集、环境、超参数、基线方法、实现细节。
2. **评价指标**：为何选这些指标、计算方式（若非常见）。
3. **结果**：主结果表/图、消融或敏感性分析、案例或可视化。
4. **分析**：结果说明、与假设的对应、局限。

## 图表与「文生图」类工具

- 若使用文生图/绘图 MCP：可生成示意图、流程图、结果图；在正文中注明「见图 X」并配简短图注。
- 表格：表头清晰、单位与显著性标注完整。

## 使用建议

- 数据与结论需与调研资料一致，避免矛盾。
- 若有对比实验，明确对比维度与结论。
